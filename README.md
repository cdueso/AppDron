# AppDron

### Aplicación de consola en .net core 2.2.

Se debe introducir tanto el área como las posiciones/órdenes de vuelo tal y como indican las especificaciones de la siguiente forma:  
  
        5 5  
        3 3 E  
        L  
        3 3 E  
        MMRMMRMRRM  
        1 2 N  
        LMLMLMLMMLMLMLMLMM
